# calipseo
Calipseo is FAO web- based platform for national Fisheries Authorities to streamline management of fisheries data and the production, analysis and reporting of fishery statistics.

Calipseo is currently managed with three software components which source code is reachable at the following repositories:
* [calipseo-model](https://github.com/un-fao/calipseo-model) Relational data model used in Calipseo to store the information
* [calipseo-app](https://github.com/un-fao/calipseo-app) Main Java (Springboot) application to collect and store information
* [calipseo-shiny](https://github.com/un-fao/calipseo-shiny). R Shiny application for Calipseo analytics (dynamic dashboards), statistical computation and reporting
